# 📝 Scratch JavaScript Todo List

A simple Todo List web application built using **HTML, CSS, and Vanilla JavaScript**.  
This project was created to understand basic **JavaScript concepts, DOM manipulation, and event handling**.

---

## 🚀 Features

- Add new tasks to the list
- Simple and clean user interface
- Beginner-friendly project
- Built completely using Vanilla JavaScript (no frameworks)

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome (for icons)

---

## 📂 Project Structure

```
project-folder
│
├── index.html      # Main HTML structure
├── style.css       # Styling for the application
└── script.js       # JavaScript logic for Todo functionality
```

---

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/todo-list.git
```

2. Open the project folder

```
cd todo-list
```

3. Open **index.html** in your browser.
